# connect4

## About

4-in-a-line ("Connect 4" by Hasbro) is a popular two-player table game played with red and yellow chips for player 1 and player 2, respectively. It is played on a 7x6 board, where the objective is to connect four of your coloured chips in a row - be that horizontal, vertical or diagonal. 

This project aims to recreate the joy of 4-in-a-line with a modern twist, allowing for solo play against the "computer" (a very basic algorithm). Both the hardware and firmware are included in this repository to allow you to create your own variants or to make the exact same one yourself!

Originally, this project was a Christmas gift for my family and friends, but I have now made it open-source so anyone can make their own, or modify it to make it even better!

## Making Your Own

If you intend to create your own, you will need - at very least - the following:

- STM Cube IDE
- Pre-compiled firmware
- Fully assembled circuit board (I recommend JLC PCB)
